# gms
## 体育馆管理系统

后台采用springboot

前端采用vue

ui库采用elementui

数据库是mysql

系统包括用户管理，赛事管理，场地管理，器材管理四个模块，采用前后端分离，实现了权限的控制。

<br>

## 使用步骤

gms文件夹是后台，可以直接用idea打开

gym-vue是是前端，依次执行

`npm install`

`npm run serve`

具体可以看gym-vue文件夹里面的readme
